如何安装 Atom Package

  • Atom
  • Editor
  • Git
  • Node.js
学习笔记

Github 推出的 Atom 编辑器，界面很友好，自带主题就已经很漂亮了，自定义功能也很强大，但是具体如何操作我还要琢磨琢磨。最重要的是扩展性，相对于 Sublime Text 来讲，差别不是很大，但比 Sublime Text 便捷了许多，它提供了官方的插件社区，可以很方便地在设置窗格进行 Packages Manage。但是令人很苦恼的一点就是，它的服务器架设在 amazonaws 上，让我等小白情何以堪啊！！！于是我就我自己折腾插件的过程，进行了一下简单总结。

直接在Atom中安装

Atom → Setting Pane → Install

直接使用apm install <package-name>命令

配置 atom、apm 环境变量

Windows系统属性 → 高级属性 → 环境变量，在“系统变量”中加入C:\Users\~\AppData\Local\atom\bin。

安装 node.js、python、git 环境

下载地址：
Node.js
Python
Git

配置网络协议

apm config set strict-ssl false
apm config set https-proxy
apm config set http-proxy

可以直接配置 socks5 协议。详情请见Behind a firewall?

通过Github安装Atom Package

使用git clone命令克隆Atom Package所在的Github repository到这个路径：~\.atom\package\，再使用apm install命令

离线安装Atom Package

直接从github下载package包，放入~\.atom\package\，通过命令行进入该文件夹路径，再使用apm install命令安装。

若apm命令尝试失败，可使用npm命令安装。
